# cctype

다음은 문자를 인자로 받아서 종류를 판별하는 함수힙니다.

| 함수명            | 판별 내용                            |
| ----------------- | ------------------------------------ |
| isalnum           | 알파벳 또는 0~9                      |
| isalpha           | 알파벳                               |
| isblank           | space 또는 tab                       |
| isdigit           | 0~9                                  |
| isxdigit          | 0~F                                  |
| islower / isupper | 소문자 / 대문자                      |
| isspace           | 빈칸                                 |
| ispunct           | 쉼표(,) 또는 마침표(.) 등의 문장기호 |

<br>

* tolower(문자) / toupper(문자)
  * 문자를 소문자 / 대문자로 변환해서 반환합니다.
