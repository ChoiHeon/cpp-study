# cmath

* **abs(실수)**
  * float, double 타입의 인자를 받아 절대값을 반환합니다.
  * 입력받은 타입에 맞는 타입으로 반환합니다.
    * float -> float
    * double -> double
    * long double -> long double
    * T -> double
  * 정수(int) 타입의 인자를 받는 abs는 \<cstdlib>에 존재합니다.

<br>

* **pow(실수, 제곱 수)**
  * 제곱한 결과를 반환합니다.

<br>

* **sqrt(실수)**
  * 실수의 루트 값을 반환합니다.
