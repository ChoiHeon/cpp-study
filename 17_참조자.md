# 참조자

참조자는 자신이 참조하는 변수를 대신하는 또하나의 이름입니다.

```c++
int n = 10;
int& r = n;
int& e = r;
int& f = e;
```

위 처럼 참조자를 대상으로 참조가 가능합니다.

<br>
다음은 에러가 발생하는 코드입니다.

```c++
int& r1 = 20;   // 상수를 참조할 수 없습니다.
int& r2;		// 선언과 동시에 초기화를 해야 합니다.
int& r3 = NULL;	// 널을 참조할 수 없습니다.

const int x = 10;
int& r4 = x;	// 참조하기 위해선 const int& r4로 작성해야 합니다.
```
